Eclipse Dynamic Languages Toolkit (DLTK) — это инструмент для поставщиков, исследователей и конечных пользователей, которые используют динамические языки. DLTK состоит из набора расширяемых фреймворков, предназначенных для упрощения создания полнофункциональных сред разработки динамических языков, таких как PHP и Perl. Помимо фреймворков DLTK предоставляет готовые к использованию интегрированные среды разработки Tcl, Ruby и Python.
